# Holberton Headphones

This project is a responsive landing page built from a Figma design.  
It focuses on building clean HTML structure, organized CSS styling, and a layout that adapts to different screen sizes.  
The goal is to reproduce the mockup as accurately as possible while keeping the code simple and readable.

![Project preview](images/readme.png)

## Features

- Fully responsive layout  
- Flexbox-based structure  
- Clean and semantic HTML  
- Organized CSS file  
- Pixel‑perfect reproduction of the Figma design  

## Technologies Used

- HTML5  
- CSS3  
- Flexbox  
- Responsive design
